# URL to Markdown Link Copy

## これは何？

ブラウザで見ているページのURLを、**ワンクリックでMarkdownリンクに変換してコピー**できるChrome拡張機能です。

ObsidianやNotionなど、Markdown記法でメモを取っている方に最適です。URLをコピーして、タイトルを入力して、リンク記法に整形して……という手間がなくなります。

## 具体例

### ✏️ 一般サイトの場合
Webページを閲覧中にアイコンを押すだけで、ページタイトルをラベルにしたリンクが生成されます。

| 閲覧中のページ | コピーされる内容 |
| :--- | :--- |
| Wikipedia「認知行動療法」のページ | `[認知行動療法 - Wikipedia](https://ja.wikipedia.org/wiki/...)` |

### 🤖 AIチャットサービスの場合
ChatGPTやGeminiなどの会話ページでは、長いページタイトルではなく**サービス名**に自動で置き換わります。

| 閲覧中のサービス | コピーされる内容 |
| :--- | :--- |
| ChatGPT | `[ChatGPT](https://chatgpt.com/c/...)` |
| Gemini | `[Gemini](https://gemini.google.com/...)` |
| Claude | `[Claude](https://claude.ai/...)` |
| Grok | `[Grok](https://grok.com/...)` |
| NotebookLM | `[NotebookLM](https://notebooklm.google.com/...)` |
| Gmail | `[Gmail](https://mail.google.com/...)` |
| Amazon | `[Amazon](https://www.amazon.co.jp/dp/B0D.../)` |

### 🛒 Amazon商品ページの場合
Amazonの長いURLから余計なトラッキングパラメータが自動で除去され、**短くきれいなURL**になります。

| 元のURL（一部省略） | コピーされる内容 |
| :--- | :--- |
| `https://www.amazon.co.jp/gp/product/B0D...?ref=...&tag=...` | `[Amazon](https://www.amazon.co.jp/dp/B0D.../)`|

### 🎓 学術論文ページの場合
学術論文のページでは、メタデータ（著者名・出版年）を自動で読み取り、**APA/日本心理学会風の引用形式リンク**を生成します。

#### 英語論文の場合
| 著者数 | コピーされる内容 |
| :--- | :--- |
| 1名 | `[Smith (2026)](URL)` |
| 2名 | `[Smith & Johnson (2026)](URL)` |
| 3名以上 | `[Smith et al. (2026)](URL)` |

#### 日本語論文の場合
| 著者数 | コピーされる内容 |
| :--- | :--- |
| 1名 | `[山田 (2026)](URL)` |
| 2名 | `[山田・田中 (2026)](URL)` |
| 3名以上 | `[山田 他 (2026)](URL)` |

## 使い方
1. URLをMarkdownリンクとしてコピーしたいサイトで、ブラウザのツールバーにある本拡張機能のアイコンをクリックします。
2. 自動的にMarkdownリンクがクリップボードにコピーされ、「✅ Done!」と表示されます。
3. お好きなエディタに貼り付けてください。

## インストール方法
1. 本リポジトリのフォルダ（`manifest.json` その他のファイルが入っているフォルダ）をPC上のどこかに配置します。
2. Chromeブラウザで拡張機能の設定を開きます（`chrome://extensions/` を開くのでもOK）。
3. 右上の「デベロッパー モード」をオンにします。
4. 左上に現れるパッケージ化されていない拡張機能を読み込む」を選択し、上記のフォルダを選択します。

## 既知の制約（今後の課題）
- **Elsevier (ScienceDirect) 等の動的レンダリングサイト**: ScienceDirect などの一部のサイトでは、JavaScriptによる動的な描画によりメタデータが直接取得できない場合があります。その場合は通常のページと同様にページタイトルがラベルとなります。
- **著者名の姓名分割**: 日本語の著者名において、姓名の間にスペース（全角・半角）がない場合は、姓と名の判別ができないためフルネームが表示されます。
